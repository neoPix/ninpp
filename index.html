NINPP

Ninpp Is Not Power Point

Par David Balan
Ceci est un commentaire

C'est quoi ?

Un framework simple de création de présentation.

  • Présentation au format HTML5
  • Le thème est en CSS
  • Javascript gère ce que CSS ne peut pas

Historique

  • Je devais faire une présentation pour mon cours de communication,
  • je ne voulais pas utiliser PowerPoint ni LibreOffice,
  • les systèmes existants ne me convenaient pas.
  • ... Donc j'ai créé mon propre système.
  • Et le voici! ☺

Ce que ça fait

  • Présentations propres et structurées,
  • enregistrement audio / vidéo,
  • progression dans la présentation,
  • gestion des évènements tactile,
  • mode présentateur,
  • animations,
  • pagination,
  • style par dispositive,
  • animation inter-dispositive,
  • gestion de l'historique.

Animations

  • Fondu,
  • glissement gauche,
  • glissement droite,
  • glissement haut,
  • glissement bas,
  • agrandissement,
  • rétrécissement,
  • rotation,
  • rotation X,
  • rotation Y.

Animations de dispositives

  • Classique,
  • glissement,
  • fondu,
  • grossissement,
  • rétrécissement.

Ce que ça fera

  • Edition WYSIWYG (ARTEKI).
    • A Real Trivial Editor Kicking It.
    • Export PDF.
  • Replay à la demande (ORTCOI).
    • Orator Replay That Circulate On Internet.

Ce que ça ne fera pas

  • Compatibilité avec les logiciels privateurs,
  • gestion avancée des animations,
  • supporter les vieux navigateurs.

Avantages

  • Pas de logiciels propriétaires,
  • extensible,
  • compatible,
  • simple,
  • léger,
  • fun.

Défaut

  • Pas terminé.
